## Como rodar localmente

Basta abrir o arquivo `index.html` no navegador — nenhuma instalação necessária.

## Como editar o conteúdo

Todo o texto está no `index.html`. Localize a seção pelo comentário correspondente:

```
<!-- HERO -->        título e especialidades
<!-- SOBRE -->       bio da psicóloga
<!-- SERVIÇOS -->    cards de cada área de atuação
<!-- ABORDAGENS -->  abordagens terapêuticas
<!-- DEPOIMENTOS --> depoimentos de pacientes
<!-- FOOTER -->      redes sociais
```

## Como publicar atualizações

```bash
git add .
git commit -m "descrição da mudança"
git push
```

O GitHub Pages atualiza automaticamente em cerca de 1 minuto.
